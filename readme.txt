A lightweight, secure, and user-friendly plugin to customize your WordPress login page logo.

== Description ==

Stupid Simple Login Logo is a lightweight plugin that allows you to easily customize your WordPress login page logo. It provides a simple, intuitive interface in the WordPress admin area to select and manage your custom login logo.

This plugin was developed specfically with freelancers, agencies, and users that have a desire to remove the default WordPress logo and set their own. No custom uploads by file managers or FTP clients, no need for page builder templates. This plugin is meant to be super simple. Upload an image from your device or select one from the WordPress Media Library. Then save. No bloated options. No hoops to jump through or URL changes. Just set and forget. Simply brand the login page for your client or agency.


= Key Features =

* Simple and intuitive admin interface
* WordPress Media Library integration
* Support for multiple image formats (JPEG, PNG, GIF, WebP, AVIF)
* Secure file handling
* No unnecessary options or settings

== Installation ==

1. Upload the `stupid-simple-login-logo` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Login Logo to configure your custom logo
4. Select your logo from the Media Library
5. Save your changes
6. Visit your login page to see the new logo

= Requirements =

* WordPress 5.8 or higher
* PHP 7.4 or higher
* Administrator access to WordPress

== Frequently Asked Questions ==

= What image formats are supported? =

The plugin supports the following image formats:
* JPEG/JPG
* PNG
* GIF
* WebP
* AVIF

= Is there a file size limit? =

Yes, the maximum file size is 5MB. This is to ensure optimal performance and security.

= Can I use any image from my Media Library? =

Yes, you can use any image from your WordPress Media Library that meets the format and size requirements.

= Will this affect my site's performance? =

No, the plugin is designed to be lightweight and has minimal impact on your site's performance.

= Is this plugin translation ready? =

Yes, the plugin is fully translation ready and includes a .pot file for translators.

= Can I preview the login page before saving? =

Yes, once you've selected a logo, you can use the "Preview Login Page" button to see how it looks.
